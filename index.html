Project Summary: Latitude vs X



This project was designed to analyze how certain weather measurements change as we change latitudes. We began by assembling a list of over 600 cities worldwide from the Open Weather Map API.

Using Matplotlib, we plotted various weather measurements vs latitude. You will see that we looked at temperature, cloudiness, wind speed, and humidity. You have the opportunity to see the source data, the visualizations created, and discussion about the trends witnessed (or not witnessed).
